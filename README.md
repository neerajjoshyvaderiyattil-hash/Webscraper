## 📋 Project Description

A powerful **General Purpose Web Scraper** built with Python using **BeautifulSoup4** and **Requests**. Scrapes any public website and saves data to both **CSV and JSON** formats. Available in **CLI, Desktop GUI (Tkinter)** and **Web App (Flask)** versions.

## ✨ Features

- 🌐 Scrape **any public website**
- 8 scraping modes for different data types
- 💾 Save results to both **CSV & JSON**
- 🖥️ GUI window with **live output console**
- 🔄 **Multi-threaded** — UI never freezes while scraping
- 📊 Live stats — Items scraped, files saved, errors
- 📁 Auto-saves to `Documents/scraped_data/` folder


## 🎯 Scraping Modes

| Mode | What it scrapes |
|---|---|
| 📄 All Text | All h1-h6, p, li, a, td tags |
| 📰 Headings | H1 through H6 headings only |
| 🔗 All Links | Every hyperlink with URL & text |
| 🖼️ All Images | All images with src & alt text |
| 📊 Tables | All HTML tables as structured data |
| 🎯 Custom CSS | Any element by CSS selector |
| 🔍 Summary | Title, word count, link/image counts |
| 🕸️ Multi-page | Scrape multiple URLs at once |


## 💡 Example Usage

# Scrape quotes from a test site
